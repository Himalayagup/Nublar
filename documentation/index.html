Welcome to Nublar

Nublar is a lightweight and powerful web framework built in Python, designed to make web development simpler and faster. Whether you're building REST APIs, serving static files, or managing dynamic content, Nublar provides a clean and intuitive way to create web applications.

Simple Routing

Define routes using function-based views (FBVs) or class-based views (CBVs) with an intuitive decorator syntax.

Built-in Templates

Powerful template rendering system with support for context data injection and template inheritance.

Cookie Management

Easy-to-use cookie handling with built-in support for setting, getting, and deleting cookies.

Static File Serving

Efficient static file handling with automatic MIME type detection and proper caching headers.

Database Integration

Simple database models with field types and automatic migrations.

Error Handling

Comprehensive error handling with customizable error pages and debug mode support.

Quick Start

Get started with Nublar in minutes:

# Install Nublar
git clone https://github.com/Himalayagup/Nublar.git
cd nublar

# Create a new app
python mediator.py startapp myapp

# Start the development server
python mediator.py runserver

For detailed instructions, check out our Getting Started guide.
